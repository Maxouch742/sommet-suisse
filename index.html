Sommets suisses

Détails

Le jeu est similaire à GeoGuessr, mais se concentre sur les montagnes en Suisse. Les joueurs doivent identifier la montagne depuis laquelle une vue 3D est prise. Ils peuvent naviguer dans la vue 3D et utiliser jusqu'à deux indices pour les aider. Une fois qu'ils pensent avoir identifié le sommet, ils cliquent sur une carte pour valider leur réponse.

Nom Score Date

Règles du jeu

Dans la vue de droite, vous pouvez observer en détails la topographie environnante du sommet. Grâce à l'environnement, vous pourrez arriver à placer le sommet sur la carte de gauche. Vous avez le droit à des indices pour vous aider. Il faut évidemment faire le meilleur score, soit la distance la plus courte au sommet.

WARNING

Votre score ne sera pas enregistré dans la base de données.
